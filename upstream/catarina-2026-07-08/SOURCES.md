# Sources

Original schema/compiler files by Catarina, retrieved 2026-07-08 from the
"Iconic CAD Village Construction Set 12 foot modules" directory linked from
[Catarina Log, July 8 2026](https://wiki.opensourceecology.org/wiki/Catarina_Log)
(Google Drive folder `1sIUagwa5A-m3XHxQ0VNfl0gXyACplKGL`, `Technical/SCHEMAS/`).

Files are preserved verbatim under their original names. The `.txt` files are
plain-text conversions of the corresponding `.rtf` files (which are RTF-wrapped
Python) for easier reading; the `.rtf` originals remain authoritative.

Converted, validated library entries derived from these files live under
`library/`; each entry's `meta.yaml` records its source file here.
